1 Посмотреть где я - pwd
2 Создать папку - mkdir Name
3 Зайти в папку	- cd name
4 Создать 3 папки - mkdir name1 name2 name3 
5 Зайти в любую папку - cd d:/testing/gitbash/1 или cd 1
6 Создать 5 файлов (3 txt, 2 json) - touch 1.txt 2.json
7 Удалить файл или папку - rm
8 Вывести список содержимого папки - ls -la
9 + Открыть любой txt файл - vim 1.txt
10 + Написать туда что-нибудь, любой текст.	 
11 + Сохранить и выйти - ctrl+x
12 Выйти из папки на уровень выше - cd ..
13 Переместить любые 2 файла, которые вы создали, в любую другую папку - mv 1.txt 4.json /d/testing/gitbash/2
либо {1.txt,4.json}
14 Cкопировать любые 2 файла, которые вы создали, в любую другую папку - cp 2.txt 5.json 3 /d/testing/gitbash/2
15 Найти файл по имени - find 1.txt - поиск в текущей директории
find /d/testing/gitbash/ -name "1.txt" - поиск в указанной директории
16 Просмотреть содержимое в реальном времени (команда grep) - cat 1.txt 
grep Hello 1.txt - пишет всю строку с эти словом
17 Вывести несколько первых строк из текстового файла - head -3 1.txt - выведет первые 3 строки из файла
18 Вывести несколько последних строк из текстового файла - tail -2 1.txt
19 Просмотреть содержимое длинного файла (команда less) - less 1.txt
20 Вывести дату и время - date
21 Выйти из vim - Esc+:+WQ+Enter

	Создаем файл script.sh, заходим в него через команду nano. Первой строкой прописываем # !/bin/bash. Ниже пишем команды по порядку.	
	Чтобы сделать файл исполняемым, прописываем команду  $ chmod ugo+x script.sh	
	Чтобы запустить скрипт, пишем ./script.sh
